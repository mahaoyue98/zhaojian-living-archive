# 照见

> 注意力，是最后一种只属于你的材质。

「照见」是一件全中文、三幕式的实时数字艺术作品。它将用户选择的心境、移动轨迹、速度与停顿，转化为持续变化的 WebGL 材质，并最终凝结成一张可保存的私人痕迹。

![照见第一幕](artifacts/照见-第一幕-感知.png)

## 三幕体验

1. **感知**：在「静观 / 好奇 / 笃定」之间选择此刻的质地。
2. **刻写**：移动指针，让速度、停顿与方向进入 GPU 反馈场。
3. **凝结**：动作路径、脉冲和场值共同生成一张独一无二的私人图像。

## 技术特点

- 原生 WebGL 珍珠材质
- GPU 双缓冲反应扩散反馈场
- 动作速度、路径与脉冲实时采样
- Web Audio 环境声状态引擎
- 甲 / 乙 / 丙三级设备能力适配
- 全中文宋体、黑体与仪器数字排版系统
- 桌面端、移动端与减少动态效果支持
- 最终作品可下载为 PNG

## 本地运行

### 一键启动

在视窗系统中双击：

```text
启动 照见.cmd
```

启动器会自动安装依赖、构建作品、启动本地服务并打开浏览器。

### 命令运行

```bash
npm install
npm run dev
```

生产构建：

```bash
npm run build
npm run preview
```

## 项目结构

```text
src/material.js    珍珠材质引擎
src/feedback.js    GPU 反馈场
src/conductor.js   声音状态引擎
src/main.js        三幕交互控制器
src/styles.css     中文视觉与字体系统
```

## 隐私

作品不上传、不保存用户动作。所有计算和图像生成都发生在本地浏览器中。
